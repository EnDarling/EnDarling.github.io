web前端学习推荐

作者：Lin 围观群众：233 更新于

28 7月 2021

本文章主要是我整理的前端学习路线，按照这个路线学习前端，一定能有充足的收获。

第一阶段，html+css+js：首先肯定是基础的html+css+js三剑客，这是最基本的知识，所以基础阶段一定要打好，这个阶段的学习时间一般是2个月以内。

第二阶段，JQuery：很多人都说jQuery已经过时了，但是起始jQuery的使用还是很广泛的，虽然很多新兴项目不使用jQuery开发了，但是维护老代码和大量的以前的老项目都大量使用了jQuery。而且基于jQuery开发的很多框架和插件也是十分强大的，所以还是需要学习jQuery，学习它的前端开发思维和处理复杂的逻辑思维的方法。

第三阶段，html5+css3：在学习了一定的html和css的基础上，我们需要学习更强大的H5和css3来强化我们的能力。这是基础部分，学完以上三个阶段后，我们就能开发完整的纯前端的带各种动画和动态效果的页面了。

第四阶段，移动端web开发：这部分是衍生出来的新板块，因为移动端开发都是基于htnl5+css3+js开发的。而且移动端开发相对于pc端的开发来说，增加了很多新的知识，会pc端的开发不一定会移动端，所以这也是我们需要学习移动端开发的原因。而且今后的发张趋势是越来越向移动端靠拢的，所以移动端才作为一个板块区分出来，重点学习。

第五阶段，es6+：其实这就是JavaScript阶段的升级阶段，我们之前学习的js全称是ECMAScript 5，是2009年发布的版本。es6全称是ECMAScript 6.0，是2015年发布的新版本，实在es5之后的大版本的更新，在es5的基础上增加了很多新的api，所以当前我们的开发主要以es5+es6一起开发的。那为什么叫es6+呢，因为现在已经发展到es6，es7，es8...的阶段了，但是7，8，9等阶段目前浏览器的支持性并不好，还没有普及，并且这些版本知识小版本的增加一些少量的api，所以不用深入学习。
